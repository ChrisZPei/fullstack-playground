# Fullstack Playground

A personal full-stack learning & project space.

**Stack (locked in):**
- Backend: FastAPI (Python) + SQLAlchemy + PostgreSQL
- Frontend: React + Next.js (TypeScript)
- Infra: Docker, Redis, GitHub Actions (CI/CD)

**Goals:**
- Learn how to build and deploy a scalable, modern full-stack app.
- Practice clean code, testing, and proper project structure.
- Build portfolio-ready projects with this stack.
